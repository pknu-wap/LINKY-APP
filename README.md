# 🔗LINKY
LINKY는 **AI 기반 링크 아카이빙 서비스**로, 
사용자가 링크를 쉽게 관리, 저장, 확인할 수 있도록 돕는 서비스입니다. 


## 핵심 기능
* **간편한 소셜 로그인 (Kakao)**
  * OAuth 2.0 기반의 카카오 로그인 기능을 통합하여 복잡한 절차 없이 서비스를 시작할 수 있습니다.
* **사용자 맞춤형 리마인더 설정**
  * 저장만 하고 잊어버리는 링크가 없도록, 사용자가 설정한 시간에 맞춰 다시 읽기 알림을 제공합니다.
* **직관적인 아카이브 관리 UI**
  * 저장된 링크들을 리스트 및 상세 보기 형태로 관리하며, 수정창을 통해 제목이나 카테고리를 자유롭게 편집할 수 있습니다.
* **AI 스마트 요약 및 자동 카테고리 분류 (예정)**
  * Gemini API를 활용하여 저장된 링크의 콘텐츠를 자동으로 분석합니다.
  * 긴 글을 읽지 않아도 핵심 내용을 파악할 수 있는 요약본과 적절한 카테고리를 자동으로 생성합니다.


## 기술 스택
* **백엔드**: Spring Boot, MySQL
* **프론트엔드**: Flutter
